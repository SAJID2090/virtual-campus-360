# virtual-campus-360
This project presents an immersive, interactive 360° Virtual Campus Navigation System designed to enhance student orientation
and career awareness through a digital experience. Built using Unity, the system integrates panoramic skybox environments,
interactive UI elements, and a multimedia-based Career Video Library. Users can explore major campus locations through a simple
navigation panel and view department-specific content such as images, videos, and 3D models. The addition of a Career Video
Library provides informative career guidance videos, accessible through a thumbnail-based interface. The system combines Unity’s
rendering pipeline, UI canvases, video playback capabilities, and modular script-based functionality to create a seamless
experience. The project demonstrates effective API-like communication between scripts, user interface, and video player
components, allowing for scalable and flexible interaction. The goal was to develop a user-friendly, visually engaging, and
technically efficient tool suitable for student orientation programs or digital campus tours. This report provides an overview of the
system architecture, implementation processes, script interaction mechanisms, and UI design considerations. The completed project
showcases a practical application of Unity for building interactive educational tools while demonstrating technical skills in
scripting, interface design, and multimedia integration.

Conclusion:


This project successfully integrates 360° media, interactive UI elements, and modular scripting to create an immersive
virtual campus navigation system. It demonstrates how Unity can be used for educational applications by combining
navigation, multimedia playback, and interactive pop-ups in a unified environment. The Career Video Library adds
further value by presenting career information in a structured and user-friendly way. The system’s modular design
allows easy expansion with new locations or media in the future. Overall, the project meets its objectives and
showcases an effective blend of technology and learning, offering users an engaging way to explore the university
campus and its academic opportunities.
